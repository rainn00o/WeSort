# WeSort

WeSort 是一个本地微信接收文件整理工具，包含去重，AI创建分类规则，文件分类功能。

## 项目简介

微信接收文件往往来源复杂、命名混乱、重复较多，长期积累后很难管理。  
WeSort 的目标是帮助你把这些文件从“堆在一个目录里”整理成“可查找、可归档、可持续维护”的本地资料库。

它适合处理这类场景：

- 微信接收的项目资料
- 图纸、文档、图片、压缩包混合目录
- 需要先去重，再分类归档的本地文件集合

## 核心功能

- 扫描本地文件目录
- 检测重复文件
- 使用 AI 辅助创建分类规则
- 按规则对文件进行分类整理
- 支持月份前缀重命名
- 支持生成分类报告

## 界面流程

1. 选择源目录
2. 扫描文件
3. 检查并处理重复文件
4. 使用 AI 辅助创建或调整分类规则
5. 执行文件分类

## 功能说明

- 本地运行，不依赖在线网盘
- 面向微信接收文件整理场景
- 支持项目分类、特殊分类和零散文件兜底
- 支持 API 设置与连通性测试
- 支持对扫描结果和重复检测结果进行可视化查看

## 适合整理的文件类型

- CAD 图纸
- PDF 文档
- Office 文档
- 图片与设计素材
- 压缩包
- 软件工具与字体文件

## 运行方式

推荐使用虚拟环境：

```powershell
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
.\.venv\Scripts\python main.py
```

## 使用建议

- 首次使用时，先扫描并查看重复检测结果，再执行移动
- 先用 AI 生成一版分类建议，再根据自己的项目结构微调规则
- 在正式整理大批文件前，建议先选择一个测试目录试跑一次

## 配置文件

项目默认配置位于 `config/` 目录下，常见文件包括：

- `config/rules.json`
- `config/api_config.json.template`

运行过程中生成的本地配置和日志文件不建议直接提交到版本库。

## 打包

当前打包配置文件：

- `WeSort.spec`

打包命令：

```powershell
.\.venv\Scripts\python -m PyInstaller WeSort.spec --noconfirm
```
